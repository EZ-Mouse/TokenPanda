# TokenPanda

TokenPanda es un addon independiente para **World of Warcraft: Mists of Pandaria Classic** que muestra dentro del juego el precio actual y el historial reciente de la Ficha de WoW.

El proyecto incluye:

- **TokenPanda Addon**: gráfico, precio, variación y alertas dentro de WoW.
- **TokenPandaTray**: microaplicación portátil para Windows que descarga y sincroniza el historial.

## Funciones principales

- Periodos 3D, 7D, 14D, 1M, 3M, 6M y ALL.
- Comparación del periodo actual con el periodo anterior.
- Modos minimizado, normal y análisis.
- Posición independiente para cada modo.
- Alertas visuales de tendencia y nuevos mínimos.
- Regiones US, EU, KR y TW.
- Interfaz en once idiomas.

## Instalación rápida

1. Cierra World of Warcraft.
2. Copia la carpeta `TokenPanda` en:

```text
World of Warcraft/_classic_/Interface/AddOns/
```

3. Ejecuta `TokenPandaTray.exe`.
4. Selecciona la carpeta de WoW y tu región.
5. Activa la sincronización.
6. Abre WoW y escribe:

```text
/tokenpanda
```

Consulta [USER_GUIDE.md](USER_GUIDE.md) para instrucciones completas.
